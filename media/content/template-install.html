Folder Structure
  • index.html - This file.
  • vamshop - Folder with templates and all assets.
  • vamshop/index.html - Main Page Template.
  • vamshop/product-listing-columns.html - Product Listing Columns Template.
  • vamshop/product-listing-list.html - Product Listing List Template.
  • vamshop/product-cart.html - Product Info Template.
  • vamshop/contact_us.html - Contact Us Page Template.
  • vamshop/shopping-cart.html - Shopping Cart Page Template.
  • vamshop/checkout.html - Checkout Page Template.
  • vamshop/assets/css - JS Files
  • vamshop/assets/components - HTML templates for components.
  • vamshop/assets/cache - Minified, combined CSS, JS files.
