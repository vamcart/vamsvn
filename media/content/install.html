System Requirements
  • Hosting: PHP + mySQL.
  • PHP: >5.6, 7.x
  • MySQL: >4.6
Installation
  1. Unzip vamshop archive.
  2. Copy files from vamshop folder to your web server folder (public_html, htdocs etc...)
  3. Open your domain name in the browser.
  4. Installation started automatically.
  5. Follow step by step instructions.
Admin Dashboard Authorization
  1. Go to My Orders menu at navbar.
  2. Click to "Login ot create new account" button.
  3. Login with your credentials. Email and password that your specified at installation process.
  4. Click Admin link at navbar.
  5. Admin dashboard available.
