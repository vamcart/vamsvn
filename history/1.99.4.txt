Опубликовано 16 июня (изменено)
Выпущена новая версия VamShop 1.99.4

Скачать полную версию VamShop 1.99.4 можно в своих заказах в магазине https://vamshop.ru
Обновление можно скачать на форуме тех. поддержки пользователей https://forum.vamshop.ru

Что изменилось:

Добавлены seo url для производителей.
Купоны и подарочные сертификаты теперь доступы для неавторизованных покупателей.
Страница корзины на ajax. Изменение количества товара в плюс, в минус, удаление товара на странице корзины. Всё это теперь через ajax, без перезагрузки страницы.
Добавлена возможность сортировки производителей, новое поле — Порядок сортировки.
Добавлен новый модуль доставки — Boxberry. С выбором пункта выдачи на карте.
Добавлен новый модуль доставки — Почта России API. Считает доставку через официальный API Почты России.
Исправлены ошибки в модуле доставки PickPoint.
Исправлены ошибки в модуле доставки EMS Почта России.
Добавлены 2 новые вкладки на странице карточки товара — с условиями доставки и с условиями оплаты. Текст условий правится в Админке — Разное — Информационные страницы.
Сортировка товаров брэнда по категориям и сортировка товаров категории по брэндам в виде dropdown по умолчанию.
Добавлена поддержка подарочных сертификатов в модуль быстрого оформления заказа.
Добавлен редирект 301 со старых страниц производителей на seo url производителей.
Добавлен логотип для html шаблонов писем.
Корректировка вывода тэгов товара на странице карточки товара.
Исправлена ошибка с отправкой подарочного сертификата на почту.
Корректировка класса rss для совместимости с новыми версиями php.
Корректировка страницы добавления отзыва, фамилия не показывается, только имя.
Добавлен 404 ответ для отсутствующих страниц в модуле новости.
Добавлен 404 ответ для отсутствующих страниц в модуле вопросы и ответы.
Корректировка шаблона vamshop1, по умолчанию выводится 4 плитки с товаром, на странице категории, т.е. там где есть левая колонка, 3 плитки в одной строке.
Кнопка пересчитать удалена со страницы корзины.
