Опубликовано 15 августа, 2018 (изменено)
Выпущена новая версия VamShop 1.92.

Скачать можно в своих заказах в магазине http://vamshop.ru/

Основные изменения:

1. Корректировка кода атрибутов товара для правильной работы на php 7.2.
2. Корректировка кода фильтров и спецификаций товара для правильной работы на php 7.2.
3. Корректировка суммы заказа в модуле оплаты Яндекс.Касса.
4. Добавлена метка PRODUCTS_STOCK для вывода на странице корзины.
5. Корректировка адреса шлюза в модуле оплаты КупитьВкредит.
6. Корректировка перевода.
7. Корректировка кода для совместимости с php 7.2.
8. Корректировка кода для совместимости с php 7.0, спасибо Kovalsky.
9. Корректировка кода в редакторе заказов, спасибо sdi_68.
10. Корректировка кода Яндекс Метрики.
11. Добавлена возможность создавать фильтры по полям длина, ширина, высота, объём.
12. Добавлены метки для вывода габаритных размеров товара в карточке товара.
