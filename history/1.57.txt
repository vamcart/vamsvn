Опубликовано 7 апреля, 2010 (изменено)
Выпущена новая версия VaM Shop 1.57.

Скачать можно в своих заказах в магазине.

Основные изменения:

1. Новый модуль оплаты - LiqPAY.

2. Новый модуль оплаты - InterKassa.

3. Добавлена автоматическая генерация SEO URL адресов для товаров, категорий, информационных страниц, категорий статей, статей, новостей, faq.

4. Добавлена карта с отметкой адреса доставки на странице просмотра заказа в админке.

5. Добавлен вывод штрих-кода в накладную.

6. Уменьшено количество SQL запросов, увеличена скорость работы магазина.

7. HTML-редактор TinyMCE включён по умолчанию, изменён способ подключения редактора, добавлен GZip компрессор.

8. Оптимизирована загрузка страницы создания/редактирования товара в админке, страница грузится гораздо быстрее.

9. JavaScript открытия картинок ThickBox заменён на FancyBox.

10. Добавлены иконки для модулей оплаты.

11. Добавлены иконки для модулей доставки.

12. Добавлен скрипт для подсветки изменённых полей формы.

13. Добавлен скрипт для автофокусировки первого поля формы.

14. Не работал поиск в faq при указании SEO URL, исправлено.

15. Исправлена ошибка в сортировке заказов по статусу в админке.

16. Исправлена ошибка в менеджере картинок TinyBrowser, неправильно определялся путь для загрузки картинок в случае установки магазина в папку.

17. Исправлена ошибка при редактировании товаров в Админке - Каталог - Скидки, не выделялся в списке отредактированный товар.

18. Исправлена ошибка в купонах, не работало ограничение купонов к конкретным товарам.

19. Исправлена ошибка в Excel импорт/экспорт, не выгружались подкатегории при выборе категории для выгрузки.

20. Добавлены css стили для форм в админке и каталоге.
