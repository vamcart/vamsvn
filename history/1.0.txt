14.6.2006, 11:05 - VamShop 1.0

Вы можете загрузить бесплатную демо-версию интернет-магазина VaM Shop.

Доступно два варианта демо-версии:
1. Готовый и настроенный пакет для установки под windows (рекомендуется для новичков).
2. Только скрипты VaM Shop для самостоятельной установки на сервер.

1. Готовый и настроенный пакет для установки под windows.
Денвер (Apache + MySQL + PHP) + установленный и настроенный VaM Shop.

Скачать можно отсюда - http://faq.oscomm.biz/vam/vamshop-demo.exe (размер 7.8Mb)

Как устанавливать:
1. Скачивайте архив http://faq.oscomm.biz/vam/vamshop-demo.exe
2. Запускайте файл и распаковывайте содержимое в любую папку.
3. Запустится установка (если не запустится установка автоматически, переходите в папку, в которую Вы распаковали файлы и запускайте файл INSTALL.bat).
4. Для продолжения нажимайте ENTER.
5. Указывайте имя директории, в которую будет установлен пакет, например C:\vamshop
6. Установить в директорию - Нажимайте y и нажимайте ENTER.
7. Для продолжения нажимайте ENTER.
8. Буквы виртуального диска, вот здесь указывайте Z и нажимайте ENTER.
9. Для продолжения нажимайте ENTER, будут скопированы файлы.
10. Затем нужно выбрать параметры конфигурации. Пишите 2 и нажимайте ENTER.
11. Создать ярлыки на рабочем столе - пишите y и нажимайте ENTER.
12. Для продолжения нажимайте ENTER.
13. Всё, магазин установлен.

Теперь на рабочем столе нажимайте ярлык Start servers.
Затем в браузере открывайте http://localhost/vamshop
Если Вы сделали всё правильно, Вы перейдёте в магазин.
Для входа в Админку, в боксе вход указывайте:
E-Mail: vam@test.com
Пароль: 12345
После входа в боксе вход появится кнопка Админка.
